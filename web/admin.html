Настройки распознавания

Система использует фиксированную конфигурацию, обеспечивающую наилучшее качество распознавания в условиях учебной аудитории.

Модель: ArcFace Детектор: RetinaFace Метрика: cosine

Максимальное косинусное расстояние между эмбеддингами лиц при котором система считает их принадлежащими одному человеку. Для ArcFace рекомендуется значение 0.68.

0.40 — строгий  |  0.68 — оптимально  |  0.80 — мягкий

Импорт расписания из Excel

Загрузите файл расписания в формате .xlsx. Система разберёт структуру файла и предложит сопоставить найденные группы с группами в базе данных.

Для каждой группы из файла выберите соответствующую группу в системе. Группы без привязки будут пропущены.
